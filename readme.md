# Pullout
Automatically buy and sell crypto. This program utilizes the scalper approach to investing, allowing crypto to bought and sold up to several times per minute (or even second). Crypto can be automatically bought when prices begin to increase and automatically sold when prices begin to decrease. In doing so, this program aims to maximize the chances of gain while minimizing the chances of loss.

## Requirements:
- python 3.13+
- ccxt
- curses
- numpy

### Additional requirements for plot tool:
- pillow
